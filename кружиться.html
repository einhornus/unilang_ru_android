кружи́ться (4373)


Etymology[edit]

кружи́ть (kružítʹ) + | -ся (-sja)

Verb[edit]

кружи́ться • (kružítʹsja) | impf

  1. to spin, to whirl, to go round, to circle
  2. passive of кружи́ть (kružítʹ)

Значе́ние[пра́вить]

  1. дви́гаться по окру́жности, кругообразно, верте́ться кру́гом, опи́сывать круговы́е движе́ния | | Кружи́ться в хорово́де. | | Плясун кру́жится на одно́й ноге́. | | Всё кружи́лось у него́ в глаза́х.
  2. враща́ться, верте́ться | | Отсу́тствует приме́р употребле́ния (см. ).
  3. беспоря́дочно передвига́ться; мета́ться | | Кружи́ться по ко́мнате, натыка́ясь на ме́бель.
  4. блужда́ть, дви́гаться, ходи́ть, е́здить, сби́вшись с доро́ги, потеря́вшись | | Кружи́ться по незнако́мым доро́гам.
  5. меша́ть, крути́ться; докуча́ть, донима́ть свои́м движе́нием, появле́нием где-либо, о́коло кого-либо | | Отсу́тствует приме́р употребле́ния (см. ).
  6. быть в постоя́нных хло́потах, суете | | Кружи́ться в хло́потах. | | Кружи́ться как белка в колесе́.
  7. взлета́ть, крути́ться в во́здухе, ви́ться в во́здухе (о сне́ге, пы́ли и | т. | п.) | | Пыль кру́жится столбо́м.
  8. де́лать, опи́сывать круги́ на ле́ту | | Орлы кру́жатся высоко в не́бе. | | Над го́родом кружи́лись самолёты.
