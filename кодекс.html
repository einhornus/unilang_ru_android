ко́декс (4754)


Noun

ко́декс • (kódɛks) m inan (genitive ко́декса, nominative plural ко́дексы, genitive plural ко́дексов)

  1. (law) code
    уголо́вный ко́декс ― ugolóvnyj kódɛks ― criminal code
    ко́декс зако́нов о труде́ ― kódɛks zakónov o trudé ― labour code
  2. code (of conduct)
    мора́льный ко́декс ― morálʹnyj kódɛks ― moral code
    ко́декс чести ― kódɛks česti ― code of honour
  3. (ancient book) codex

ко́-декс

Существи́тельное, неодушевлённое, мужско́й род, 2-е склонение (тип склоне́ния 1a по классифика́ции А. А. Зализняка).

Ко́рень: -кодекс- | Тихонов, 1996 | .

Значе́ние

  1. истор. древнеаккадийский строи́тельный те́рмин, означа́вший укла́дку камней (или слоёв камней) оди́н на друго́го. В ча́стности, кирпи́чная кла́дка — одна́ из разнови́дностей ко́декса | Отсу́тствует приме́р употребле́ния (см. рекомендации).
  2. истор. в анти́чной и средневеко́вой культу́ре: рукопи́сная книга из разре́занных и сброшюрованных листо́в | О́коло I в. сви́тки бы́ли вы́теснены ко́дексами.
  3. юр. еди́ный систематизи́рованный законода́тельный акт, объединя́ющий правовы́е но́рмы, относя́щиеся к еди́ной о́трасли права в какой-либо стране́ | Текст Трудово́го ко́декса даётся в реда́кции Федера́льных зако́нов.
  4. перен. совоку́пность правил | Ко́декс поведе́ния на са́йте.

Сино́нимы

  1. —
  2. свод зако́нов
  3. правила
