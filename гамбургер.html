га́мбургер (25123)


Noun

га́мбургер • (gámburger) m inan (genitive га́мбургера, nominative plural га́мбургеры, genitive plural га́мбургеров)

  1. hamburger (sandwich), beefburger, burger
  2. (graphical user interface) hamburger icon, hamburger button (an icon with three horizontal lines, indicating a menu in graphical user interfaces)

Usage notes

  • The graphical user interface sense is usually written with quotation marks, e.g. ико́нка «га́мбургер».

Related terms

  • чи́збургер (čízburger)

га́м-бур-гер

Существи́тельное, неодушевлённое, мужско́й род, 2-е склоне́ние (тип склоне́ния 1a по классифика́ции А. А. Зализняка).

Ко́рень: -гамбургер-.

Значе́ние

  1. вид сэндвича, состоя́щий из пригото́вленного фа́рша в приплюснутой фо́рме, по́данного внутри́ разре́занной бу́лки
    Са́мый большо́й в ми́ре га́мбургер включён в меню́ одно́й из пивны́х Пенсильвании (Сша). «Кунсткамера» // «Нау́ка и жизнь», 2008 г.
  2. перен., комп. жарг. ико́нка, состоя́щая из трёх паралле́льных горизонта́льных ли́ний, кото́рая испо́льзуется для изображе́ния кно́пки меню́ в графи́ческом пользовательском интерфе́йсе
    Три чёрные поло́ски на экра́не, напомина́ющие га́мбургер, привы́чны и знако́мы всем, кто по́льзуется цифровы́ми интерфе́йсами.
