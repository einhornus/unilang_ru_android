group (139)


ед. ч. мн. ч.
group groups

Существи́тельное

Значе́ние

  1. гру́ппа (люде́й, предме́тов)
    • There is a group of houses behind the hill. — Э́то гру́ппа домо́в за холмо́м.
  2. гру́ппа; группиро́вка, фра́кция; анса́мбль
    • Did you see the new jazz group? — Ты ви́дел но́вую джа́зовую гру́ппу?
  3. гру́ппа, класс
    4. хим. столбе́ц в периоди́ческой табли́це хими́ческих элеме́нтов
  4. хим. радика́л
  5. матем. гру́ппа
    6. военн. структу́ра в военно-воздушных си́лах комме́рческая организа́ция

Сино́нимы

  1. collection, set
  2. band, ensemble

Значе́ние

  1. группировать, образо́вывать гру́ппу
    • Group the elements by name. — Группировать элеме́нты по и́мени.
  2. группироваться, собира́ться
  3. подбира́ть, располага́ть
  4. гармони́ровать, сочета́ться
  5. мед. определя́ть гру́ппу крови

Сино́нимы

  1. amass, categorize, classify, collect, collect up, gather, gather together, gather up
