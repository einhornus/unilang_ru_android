пуска́ть (1463)


Verb[edit]

пуска́ть • (puskátʹ) | impf (perfective пусти́ть)

  1. to let go, to relinquish
  2. to set going, to set in motion, to set in operation, to start, to set working
  3. to let in
  4. to launch, to throw
  5. to sprout, to put out, to put forth
