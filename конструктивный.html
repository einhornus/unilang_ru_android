конструкти́вный (10875)


Etymology[edit]

From конструктив (konstruktív) + | -ный (-nyj), ultimately from Latin cōnstructīvus, from Latin cōnstructus (“constructed”).

Adjective[edit]

конструкти́вный • (konstruktívnyj)

  1. constructive (helpful)
    конструкти́вная кри́тика ― konstruktívnaja krítika ― constructive criticism
  2. (relational) construction, constructive

Морфологи́ческие и синтаксические сво́йства

конс-трук-тив-ный

Прилага́тельное, тип склоне́ния по классифика́ции А. | Зализняка | — 1*a.

Ко́рень: -конструк-; суффиксы: -т-ивн; оконча́ние: -ый | Тихонов, 1996 | .

Значе́ние

  1. относя́щийся к констру́кции, необходи́мый для конструи́рования | | Конструкти́вные фо́рмы материа́ла. | | Конструкти́вный недоста́ток.
  2. создаю́щий осно́ву для дальне́йшей рабо́ты; плодотво́рный | | Конструкти́вная кри́тика. | | Конструкти́вные предложе́ния.
  3. свя́занный с прямы́м получе́нием ну́жного математи́ческого объе́кта, а не то́лько с доказа́тельством того́, что он есть | | Конструкти́вное доказа́тельство.
