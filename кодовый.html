ко́довый (12726)


Etymology

код (kod) + | -овый (-ovyj)

Adjective

ко́довый • (kódovyj)

  1. (relational) code
    ко́довое и́мя ― kódovoje ímja ― code name
    ко́довая страни́ца ― kódovaja straníca ― code page

  • ко́декс m (kódɛks)
  • декоди́ровать impf or pf (dɛkodírovatʹ)
