водоро́дный (18025)


Etymology

водоро́д (vodoród) + | -ный (-nyj)

Adjective

водоро́дный • (vodoródnyj)

  1. (relational) hydrogen; hydrogenous
    водоро́дная бо́мба ― vodoródnaja bómba ― hydrogen bomb

Related terms

  • водоро́д (vodoród)
  • вода́ (vodá)
  • род (rod), родно́й (rodnój)
  • рожа́ть impf (rožátʹ), рожда́ть impf (roždátʹ), роди́ть pf (rodítʹ)

Прилага́тельное

Значе́ние

  1. свя́занный, соотнося́щийся по значе́нию с существи́тельным водоро́д
  2. сво́йственный водоро́ду, характерный для него́
  3. содержа́щий в своём хими́ческом соста́ве водоро́д
  4. испо́льзующий водоро́д как термоя́дерное то́пливо
  5. производи́мый с по́мощью водоро́да
  6. устар. рожда́ющий во́ду
    • Орле преславный, высоко парящий, // Сла́вою орлы вся превосходя́щий, // Вскую превы́ше облак водоро́дных // Париши крилы перий славоплодных?
  7. устар. рождённый в воде, из воды
    • Чу́да морски́е ещё и страши́лища все водоро́дны // Бездн из вла́жных свои́х и глубо́ких тех исходи́ли...

Сино́нимы

  4. термоя́дерный
