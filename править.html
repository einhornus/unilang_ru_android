править (6426)


Etymology

пра́вый (právyj) + | -ить (-itʹ)

Verb

пра́вить • (právitʹ) impf

  1. to govern, to rule
  2. to drive, to steer
  3. to correct
  4. to strop, to set
  5. (computing) to edit, i.e. to change the content of a file, a website or the code of software

Related terms

  • попра́вка f (poprávka)
  • пра́вило n (právilo), прави́ло n (pravílo)
  • прави́тельство n (pravítelʹstvo)
  • прави́тель m (pravítelʹ), прави́тельница f (pravítelʹnica)
  • пра́вка f (právka)
  • правле́ние n (pravlénije)
  • пра́во n (právo)
