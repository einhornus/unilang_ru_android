исся́кнуть (11132)


Etymology[edit]

ис- (is-) + | сякнуть (sjáknutʹ)

Verb[edit]

исся́кнуть • (issjáknutʹ) | pf (imperfective сякнуть or иссяка́ть)

  1. to run dry, to dry up
  2. to run out, to disappear, to fail, to run low, to run short
