арха́ровец (27489)


ар-ха́-ро-вец

Существи́тельное, одушевлённое, мужско́й род, 2-е склонение (тип склоне́ния 5*a по классифика́ции А. А. Зализняка).

Ко́рень: -архаров-; суффикс: -ец | Тихонов, 1996 | .

  1. истор. разг. аге́нт поли́ции (в дореволюцио́нной России) | 1 | | 1 | | 1 | | ▼ 1 | | Отсу́тствует приме́р употребле́ния (см. рекомендации).
  2. перен. непоседа, шалун, озорник | 2 | | 2 | | 2 | | ▼ 2 | | И бы́ло э́то заведе́ние вро́де звери́нца: арха́ровцы, скандали́сты, обло́мы; все как на подбо́р — са́мые развращённые мальчи́шки. А. И. Куприн, «С у́лицы», 1904 г. (цита́та из Национа́льного корпуса ру́сского языка́, см. Спи́сок литерату́ры)
  3. перен. отча́янный, беспу́тный челове́к; озорник, буя́н, головоре́з | 3 | | 3 | | 3 | | ▼ 3 | | Пошёл в це́рковь, наро́ду тьма-тьмущая. У двери стоят арха́ровцы с большу́щими ножа́ми и то и дело толка́ют наро́д да бью́т их кулака́ми. Ф. М. Решетников, «Никола Знаменский», 1866 г. (цита́та из Национа́льного корпуса ру́сского языка́, см. Спи́сок литерату́ры)
  4. прост. жу́лик, моше́нник, вор | 4 | | 4 | | 4 | | ▼ 4 | | Я уж не зна́ю, что мне с э́тими арха́ровцами и де́лать.
