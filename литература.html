литерату́ра (885)


Etymology[edit]

Borrowed from Latin litteratura.

Noun[edit]

литерату́ра • (literatúra) | f | inan (genitive литерату́ры, nominative plural литерату́ры, genitive plural литерату́р)

  1. literature

Related terms[edit]

  • литера́тор | m (literátor)
  • литерату́рный (literatúrnyj)
  • литературове́дение | n (literaturovédenije)
  • литературщина | f (literatúrščina)

Морфологи́ческие и синтаксические сво́йства[пра́вить]

ли-те́-ра-ту́-ра

Существи́тельное, неодушевлённое, же́нский род, 1-е склоне́ние | (тип склоне́ния 1a по классифика́ции А. | А. | Зализняка).

Ко́рень: -литерат-; суффикс: -ур; оконча́ние: -а | Тихонов, 1996 | .

Значе́ние[пра́вить]

  1. де́ятельность, свя́занная с сочине́нием те́кстов | | Занима́ться литерату́рой. | | Урок по литерату́ре. | | Пи́сьменная литерату́ра. | | Бесписьменная литерату́ра.
  2. проду́кт тако́й де́ятельности, пи́сьменные или у́стные те́кстовые произведе́ния | | Ру́сская литерату́ра. | | Класси́ческая литерату́ра. | | Худо́жественная литерату́ра. | | Техни́ческая литерату́ра. | | Спра́вочная литерату́ра.
  3. совоку́пность публика́ций по какой-либо специа́льной те́ме | | Литерату́ра по эконо́мике.

Сино́нимы[пра́вить]

  1. сочини́тельство, письмо́, писа́тельство, беллетри́стика
  2. текст
  3. библиогра́фия
