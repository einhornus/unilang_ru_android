петля́ть (12042)


Etymology[edit]

Perhaps from петля (pétlja) + | -ять (-játʹ).

Verb[edit]

петля́ть • (petljátʹ) | impf

  1. (colloquial) to dodge

Related terms[edit]

  • петля (pétlja), пе́телька (pételʹka)
  • петлистый (petlístyj)
  • петли́ца (petlíca)

Значе́ние[пра́вить]

  1. Идти́, дви́гаться не по прямо́й ли́нии, де́лая пе́тли. | | За́яц петляет. | | Петля́ть в ле́су. | | Петля́ть по двора́м.

  3. лекс., в зн. прил. | | Петляющая похо́дка. | | Петляющая доро́га.
  4. Говори́ть сбивчиво, путано, не доходя́ до существа́ дела или уклоня́ясь от гла́вного. | | Петля́ть при отве́те на вопро́с экзамена́тора. | | Петля́ть в разгово́ре. | | Петля́ть вокру́г да о́коло. | | Петля́ть и ничего́ толко́вого не сказа́ть.
