забега́ть (6118)


Etymology 1

за- (za-) + | бе́гать (bégatʹ)

Verb

забе́гать • (zabégatʹ) pf

  1. to start running
  2. to become shifty (of a look)

Etymology 2

за- (za-) + | бе́гать (bégatʹ) + | -а́ть (-átʹ)

Verb

забега́ть • (zabegátʹ) impf (perfective забежа́ть)

  1. to get (в (v) + accusative or на (na) + accusative somewhere) by running or as if by running
    Он ка́ждый день забега́ет туда́ пе́ред рабо́той.
    On káždyj denʹ zabegájet tudá péred rabótoj.
    He runs that way every day before work.
  2. to run ahead (in anticipation), to run before someone
    Не на́до забега́ть вперёд!
    Ne nádo zabegátʹ vperjód!
    No need to run ahead!
  3. (colloquial) to visit, to drop in (к (k) on)

за-бе́-гать

Глаго́л, совершенный вид, непереходный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующего глаго́ла несоверше́нного ви́да нет.

Приста́вка: за-; ко́рень: -бег-; суффикс: -а; глагольное оконча́ние: -ть.

Значе́ние

  1. нача́ть бегать

забега́ть

Глаго́л, несоверше́нный вид, непереходный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л совершенного ви́да — забежа́ть.

Приста́вка: за-; ко́рень: -бег-; суффикс: -а; глагольное оконча́ние: -ть.

Значе́ние

  1. передвига́ясь бе́гом, попада́ть куда-либо
    Он ка́ждый день забегает туда́ пе́ред рабо́той.
  2. перен. передвига́ясь бе́гом, обгоня́ть или кого-либо, ока́зываться впереди́ или сбо́ку
    Не на́до забега́ть вперёд!
