столо́вый (1862)


Etymology[edit]

стол (stol) + | -овый (-ovyj)

Adjective[edit]

столо́вый • (stolóvyj)

  1. (relational) table
    столо́вая ло́жка ― stolóvaja lóžka ― spoon
  2. (relational) dinner
    столо́вый набо́р ― stolóvyj nabór ― dinner service

Related terms[edit]

  • стол | m (stol)
  • столо́вая | f (stolóvaja)

Морфологи́ческие и синтаксические сво́йства[пра́вить]

сто-ло-вы́й

Прилага́тельное, тип склоне́ния по классифика́ции А. | Зализняка | — 1a.

Ко́рень: -стол-; суффикс: -ов; оконча́ние: -ый.

Значе́ние[пра́вить]

  1. употребля́емый за столо́м, во вре́мя обе́да, у́жина | | Столо́вая ло́жка. | | Столо́вое серебро́. | | Столо́вый прибо́р. | | Столо́вое бельё. | | Столо́вое вино́. | | Столо́вая соль.
  2. предназна́ченный, употребля́емый, расхо́дуемый на пита́ние | | Столо́вые деньги. | | Столо́вые расхо́ды.
  3. сво́йство по значе́нию прилага́тельного стол | | Столо́вая но́жка.
