гото́вить (1544)


Etymology[edit]

Synchronically analysable as гото́вый (gotóvyj) + | -ить (-itʹ).

Verb[edit]

гото́вить • (gotóvitʹ) | impf (perfective пригото́вить or подгото́вить or загото́вить)

  1. to prepare, to make ready
    Perfective: пригото́вить (prigotóvitʹ)
  2. to cook
    Perfective: пригото́вить (prigotóvitʹ)
  3. to prepare, to train
    Perfective: подгото́вить (podgotóvitʹ)
  4. to store up, to lay in (stock)
    Perfective: загото́вить (zagotóvitʹ)

Synonyms[edit]

  • подгота́вливать (podgotávlivatʹ)

Related terms[edit]

  • готовка (gotóvka)
  • гото́вый (gotóvyj)
  • загото́вка (zagotóvka)
  • подгото́вка (podgotóvka)
  • гото́виться (gotóvitʹsja), пригото́виться (prigotóvitʹsja), подгото́виться (podgotóvitʹsja), заготовиться (zagotóvitʹsja)

Значе́ние[пра́вить]

О́бщее прототипическое значе́ние — приводи́ть в соотве́тствие с за́мыслом или ожидаемым собы́тием.

  1. приводи́ть в соотве́тствие с ожидаемыми результа́тами, де́лать гото́вым, то́ есть приго́дным для чего-либо | | Тре́нер гото́вит своего́ подопе́чного к соревнова́ниям.
  2. выполня́ть, составля́ть, создава́ть что-либо | | Шко́льник гото́вит уро́ки. | | Теа́тр гото́вит очередну́ю премье́ру.
  3. держа́ть, храни́ть что-либо в ожида́нии какого-либо собы́тия.
  4. замышля́ть, организо́вывать | | Враги́ гото́вили ему́ западню́.
  5. та́кже неперех. создава́ть ку́шанья, стря́пать, занима́ться кулинарией | | По́вар гото́вит рагу́ из свини́ны. | | Моя ма́ма хорошо́ гото́вит.
