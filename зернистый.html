зерни́стый (27268)


Etymology

зерно́ (zernó) + | -и́стый (-ístyj)

Adjective

зерни́стый • (zernístyj) (comparative (по)зерни́стее or (по)зерни́стей)

  1. grainy, granular

Прилага́тельное

Значе́ние

  1. состоя́щий из зёрен, содержа́щий мно́го зёрен; бога́тый зёрнами
    • Вы́ше пояса // Рожь зерни́стая // Дремит колосом // Почти́ до земи, // Сло́вно бо́жий гость, // На все стороны // Дню весёлому // Улыба́ется.
    • Вы́лила ты из по́та на́шего колосья зерни́стые, кро́вью нашей напои́ла я́годы свои́.
    • Но она ло́вко положи́ла ему́ на стекля́нную таре́лочку чуде́сного мали́нового варе́нья, и сра́зу поде́йствовала э́та кле́йкая, ослепи́тельно кра́сная сла́дость, кото́рая зерни́стым огнём перелива́лась на языке́, души́стым са́харом облипала зу́бы.
  2. состоя́щий из отде́льных крупи́нок, из зерновидных части́ц
    • За пунькой, на ска́те косого́ра, покры́том зерни́стым сне́гом, он постоя́л, ду́мая о Лю́бке.
    • Зерни́стая икра́ стоя́ла в пяти́ сере́бряных ведёрках вазонах по четы́ре фу́нта.
    • Дверь отвори́лась, и седо́й монах-привратник впусти́л их в у́зкий и высо́кий подзе́мный ход, проры́тый уже́ не в рассы́пчатом, а в зерни́стом туфе, доста́точно ры́хлом для удо́бного прокапывания галере́й.
  3. шерохова́тый, с ме́лкими неро́вностями
    • Зерни́стый ка́мень на́бережных и зда́ний блести́т крупи́нками слюды, напомина́ющими большие бры́зги.

Сино́нимы

  1. частичн. зерново́й, крупяной
  2. зернёный, гранулярный
  3. шерохова́тый
