Существи́тельное

Значе́ние

  1. разг. де́йствие по значе́нию гл. врать
    • Пять челове́к — Враль, // Трупиков, // Отпе́тый, // Мерзавкер и Жульё — приле́жно, // день за днём, // Публи́чным занима́ются враньём // И вку́пе называ́ются г а з е т о й.
  2. разг. результа́т тако́го де́йствия, то есть ложь; вы́думка, вздор
    • — Никаки́х уро́дов на све́те нет, — не о́чень уве́ренно сказа́л Кандид. — Враньё э́то всё…

Сино́нимы

  1. —
  2. ложь, вы́думка, вздор, небыли́ца, непра́вда; частичн.: неле́пость
