раздолба́ть (18345)


Etymology[edit]

раз- (raz-) + | долба́ть (dolbátʹ)

Verb[edit]

раздолба́ть • (razdolbátʹ) | pf

  1. (low | | colloquial) to break using multiple blows
  2. (low | | colloquial) to destroy by using sloppily
  3. (low | | colloquial) to scold, to criticize
