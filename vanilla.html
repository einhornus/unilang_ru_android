vanilla (11877)


ед. ч. мн. ч.
vanilla vanillas

Существи́тельное

Значе́ние

  1. ботан. ваниль (расте́ние, Vanilla)
  2. ваниль (пря́ность)
  3. разг. ванильное моро́женое
  4. разг. красо́тка

Значе́ние

  1. гастрон. ванильный
  2. разг. просто́й, про́стенький, станда́ртный; обыкнове́нный, неизменённый
    • vanilla Linux Kernel — ядро́ Linux, со́бранное из исходников, иденти́чных тем, что лежа́т на его́ официа́льном са́йте kernel.org
    • страница на VanillaJS — страница, сде́ланная с испо́льзованием возмо́жностей стандартизированного JavaScript вме́сто испо́льзования фреймворков вро́де JQuery, Angular, React или Bootstrap
