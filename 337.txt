<p><strong class="Latn headword" lang="en">font</strong> (<i>plural</i> <b class="Latn form-of lang-en p-form-of" lang="en">fonts</b>)
</p>
<ol><li><span class="ib-brac">(</span><span class="ib-content">typography</span><span class="ib-brac">)</span> A set of glyphs of unified design, belonging to one typeface (e.g., Helvetica), style (e.g., italic), and weight (e.g., bold). Usually representing the letters of an alphabet and its supplementary characters.
<ol><li>In metal typesetting, a set of type sorts in one size.</li>
<li>In phototypesetting, a set of patterns forming glyphs of any size, or the film they are stored on.</li>
<li>In digital typesetting, a set of glyphs in a single style, representing one or more alphabets or writing systems, or the computer code representing it.</li></ol></li>
<li><span class="ib-brac">(</span><span class="ib-content">computing</span><span class="ib-brac">)</span> A computer file containing the code used to draw and compose the glyphs of one or more typographic fonts on a computer display or printer.</li></ol>
