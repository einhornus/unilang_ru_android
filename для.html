для (41)


Etymology[edit]

From Proto-Slavic *dьľa, *děľa. Related to делить (delítʹ, “to share, divide”) | ?

Preposition[edit]

для • (dlja) (+ genitive case)

  1. for
    для детей ― dlja detéj ― for the children

Participle[edit]

для • (dlja)

  1. (dated) present imperfective adverbial participle of длить (dlitʹ)

Морфологические и синтаксические свойства

для

Предлог; неизменяемое слово. Употребляется с родительным п.

Корень: -для-.

Встречается также вариант написания | (в качестве сокращения): д/.

Значение

  1. имея целью сделать что-то тако́е, что полезно и́ли приятно друго́му челове́ку и чего́ э́тому челове́ку нормально хоте́ть | | Сделай э́то для меня. | | Что я могу́ для Вас сделать? | | Няня готовит и стирает для ребёнка. | | Она́ испекла для сына пироги.
  2. описывает цель, предназначение чего-либо | | Он много сделал для победы. | | Для э́того мы и пишем свои́ статьи. | | Он приехал для развития местного бизнеса. | | Прибор предназначен для измерения энергии.
  3. обозначает субъект, на кото́рый направлено какое-либо действие и́ли кото́рый является адресатом чего-либо либо воспринимает что-то | | Для меня мир на какое-то вре́мя остановился.
  4. указывает на сравнение свойств объекта со свойствами класса, к кото́рому этот объект относится | | Для двухгодовалого ребёнка этот малыш о́чень силён.

Синонимы

  1. ради1, на1.14, за1.10, во имя, на благо, во благо, в пользу
  2. ради2, из-за5, наI.11, вI.8, заI.10, во имя2, в целях, с целью
  3. относительно
  4. по сравнению с
