ложно (31174)


Etymology[edit]

ло́жный (lóžnyj) + | -о (-o)

Adverb[edit]

ложно • (lóžno)

  1. falsely (in a false manner)

Adjective[edit]

ложно • (lóžno)

  1. short neuter singular of ло́жный (lóžnyj)

Морфологи́ческие и синтаксические сво́йства[пра́вить]

ложно

Наре́чие; неизменяемое.

Ко́рень: --.

Значе́ние[пра́вить]

I нареч. к ло́жный ка́честв.

  1. не соотве́тствуя и́стине, бу́дучи оши́бочным, ненастоя́щим. | | Ложно истолкова́ть фа́кты. | | Отсу́тствует приме́р употребле́ния (см. ).
  2. фальши́во, притво́рно, показно́. | | Отсу́тствует приме́р употребле́ния (см. ).
  3. несправедли́во | | Жа́ль пацано́в, кото́рых ложно обвини́ли, но больше всего́ мне жа́лко Валевиду!

II

  1. оценочная характери́стика чего-либо как не соотве́тствующего и́стине, оши́бочного, не настоя́щего. | | [Арманда] по а́ктам ложно зна́чится до́черью Марии Эрве. (Булгаков, Жизнь господи́на де Мольера) | | Отсу́тствует приме́р употребле́ния (см. ).
  2. оценочная характери́стика чего-либо как фальши́вого, притво́рного, показно́го. | | Отсу́тствует приме́р употребле́ния (см. ).
  3. оценочная характери́стика чего-либо как имеющего лишь вне́шнее схо́дство с настоя́щим, действи́тельным. | | Отсу́тствует приме́р употребле́ния (см. ).

Сино́нимы[пра́вить]

  1. непра́вильно, неве́рно, по оши́бке, искажённо, шиворот-навыворот; мнимо, притво́рно, несправедли́во, искаженно, фикти́вно, оши́бочно, неточно, превратно, облыжно
