раската́ть (26619)


Etymology[edit]

рас- (ras-) + | ката́ть (katátʹ)

Verb[edit]

раската́ть • (raskatátʹ) | pf (imperfective раска́тывать)

  1. to roll (out)
    раската́ть губы ― raskatátʹ guby ― to feel one's mouth watering (for), get ready to grab, think that smth is up for grabs
  2. to make slippery (by sliding on)
  3. to roll apart, (a log building) to take apart

Related terms[edit]

  • ката́ть (katátʹ)
  • раска́т (raskát)
  • раска́тистый (raskátistyj)
  • раскатываться (raskatyvatʹsja), раскататься (raskatatʹsja), раскатиться (raskatitʹsja)

Значе́ние[пра́вить]

  1. разверну́ть что-либо скатанное | | Раската́ть сви́ток.
  2. сделать ско́льзким, гла́дким, укатанным | | Тра́ктора раската́ли доро́гу.
  3. путём ката́ния кру́глого предме́та поверху чего-либо разгла́дить, сделать пло́ским, ро́вным | | Раската́ть тесто скалкой по столу́.
  4. путём качения распредели́ть по сторона́м что-либо ра́нее со́бранное вме́сте | | Раската́ть брёвна по все́й площа́дке.
