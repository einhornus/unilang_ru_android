набе́гать (11920)


Etymology 1[edit]

на- (na-) + | бегать (bégatʹ)

Verb[edit]

набе́гать • (nabégatʹ) | pf

  1. (colloquial) to cover a total (distance)
  2. (colloquial) to run (till one is), to run oneself (to the point of)
    набе́гать себе́ аппети́т
    nabegatʹ sebé appetít
    to get a good appetite after much running
    набе́гать себе́ оды́шку
    nabegatʹ sebé odýšku
    to run till one is short of breath
Related terms[edit]
  • набегаться (nabegatʹsja)

Etymology 2[edit]

на- (na-) + | бегать (bégatʹ) + | -ать (-átʹ)

Verb[edit]

набе́гать • (nabegátʹ) | impf (perfective набежа́ть)

  1. to run (against), to dash
  2. (waves) to roll in, to coast in
  3. to run (over), to cover
    бу́квы набе́гают одна́ на другу́ю
    búkvy nabegajut odná na drugúju
    letters are running over
  4. to come running (together)
  5. (liquid) to accumulate (quickly)
  6. (cloth) to ruck up
Related terms[edit]
  • набе́г (nabég)
