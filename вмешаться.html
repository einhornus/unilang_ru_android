вмеша́ться (3953)


Etymology

в- (v-) + | меша́ть (mešátʹ) + | -ся (-sja)

Verb

вмеша́ться • (vmešátʹsja) pf (imperfective вме́шиваться)

  1. to interfere, to intervene, to meddle

Related terms

  • вмеша́тельство (vmešátelʹstvo)

Глаго́л

Значе́ние

  1. приня́ть уча́стие в чём-либо без приглаше́ния, ча́сто с це́лью пресече́ния нежела́тельных после́дствий
    • В. в чужо́й разгово́р.
    • В. в дра́ку.
    • В дело вмеша́лся суд.
    • К тому ж — он мы́слит — в э́то дело // Вмеша́лся ста́рый дуэлист; // Он зол, он спле́тник, он речист…
    2. разг., редк. прони́кнув в большую гру́ппу, затеря́ться в ней устар. вступи́ть в круг танцу́ющих, в хорово́д
    • Когда́ под зво́нкие напе́вы, // Под звук свире́ли плясовой, // Среди́ поле́й, рука́ с руко́й, // Кружатся ю́ноши и де́вы, // Вмеша́вшись в ре́звый хорово́д // Хариты, ве́треный Эрот, // Дриады, Фавны пля́шут с ни́ми // И го́нят прочь толпу́ забо́т // Воскликновеньями свои́ми…
    • Я помню, я помню друго́е свида́нье: // На ба́ле блестя́щем, в кипя́щем собранье, // Гордя́сь кавале́ром, и об ру́ку с ним, // Вмеша́лась я в та́нцы... и сча́стьем мои́м // В тот вечер прекра́сный весь мир озлащался.

Сино́нимы

  1. ввяза́ться, су́нуться, част.: втесаться, втере́ться
  2. замешаться, смеша́ться, затеря́ться
  3. вступи́ть
