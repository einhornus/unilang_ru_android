сбега́ть (4556)


Etymology

с- (s-) + | бе́гать (bégatʹ) + | -а́ть (-átʹ)

Verb

сбега́ть • (sbegátʹ) impf (perfective сбежа́ть)

  1. to run down (from)
    сбега́ть с ле́стницы ― sbegátʹ s léstnicy ― to run downstairs
    сбега́ть по ле́стнице ― sbegátʹ po léstnice ― to run downstairs
  2. to run away, to make off, to flee, to break out
  3. to escape
  4. to elope

Verb

сбе́гать • (sbégatʹ) pf

  1. (colloquial) to run (to a place) and get back, to run (for), to go and get
    сбе́гайте за врачо́м. ― Sbégajte za vračóm. ― Run for a doctor.
    Мне ну́жно сбе́гать в магази́н. ― Mne núžno sbégatʹ v magazín. ― I need to make a quick run over to the shop.

сбе́-гать

Глаго́л, совершенный вид, непереходный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л несоверше́нного ви́да — бегать.

Приста́вка: с-; ко́рень: -бег-; суффикс: -а; глагольное оконча́ние: -ть.

Значе́ние

  1. бы́стро добра́ться куда-либо бе́гом с какой-либо це́лью и верну́ться обра́тно | Мне на́до сбега́ть за хле́бом.

Сино́нимы

сбе-га́ть

Глаго́л, несоверше́нный вид, непереходный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л совершенного ви́да — сбежа́ть.

Приста́вка: с-; ко́рень: -бег-; суффикс: -а; глагольное оконча́ние: -ть.

Значе́ние

  3. соверша́ть побе́г | А в основно́м, де́ти хотя́т жить свое́й жи́знью и ни от кого́ не зави́сеть, но все равно́ для э́того на́до снача́ла поду́мать, а потом уже́ сбега́ть из дома.
