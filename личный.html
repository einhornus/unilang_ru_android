ли́чный (412)


Etymology[edit]

From лицо́ (licó) + | -ный (-nyj).

Adjective[edit]

ли́чный • (líčnyj)

  1. personal (relating to a particular person)
  2. (grammar) personal
    ли́чное местоимение ― líčnoje mestoiménije ― personal pronoun

Synonyms[edit]

  • персона́льный (personálʹnyj), ча́стный (částnyj), со́бственный (sóbstvennyj)
  • инти́мный (intímnyj)

Морфологи́ческие и синтаксические сво́йства

лич-ный

Прилага́тельное, относи́тельное, тип склоне́ния по классифика́ции А. | Зализняка | — 1*a.

Ко́рень: -лич-; суффикс: -н; оконча́ние: -ый | Тихонов, 1996 | .

Значе́ние

  1. осуществля́емый при непосре́дственном уча́стии челове́ка. | | Ли́чное уча́стие в боевы́х де́йствиях. | | Ли́чное прису́тствие на церемони.
  2. принадлежа́щий самому челове́ку, его́ со́бственный | | Ли́чная со́бственность. | | Ли́чные ве́щи.
  3. глубоко прочу́вствованный, свя́занный с со́бственными мы́слями и пережива́ниями челове́ка | | Э́то бы́ло что-то о́чень ли́чное.
  4. свя́занный с граммати́ческой катего́рией лица | | Ли́чные фо́рмы глаго́лов.

Сино́нимы
